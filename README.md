Dreadful Designs Agency 

Dreadful Designs Agency is a frontend website that showcases our dedication, style, and uniqueness in web development. This project represents our passion for design and development, merging creativity with technology.

🚀 Features

Elegant and modern UI/UX design

Fully responsive layout

Smooth animations and interactions

Showcasing our design philosophy and approach

🛠️ Technologies Used

HTML - Structuring the web pages

CSS - Styling the components

JavaScript - Adding interactivity

Tailwind CSS - Rapid UI development

📂 Installation & Setup

Clone the repository:

git clone https://github.com/Swaroopnaik885/Dreadful-Designs-Agency.git

Navigate to the project folder:

cd dreadful-designs-agency

Open index.html in your browser.
